# ERA-Engine (Godot) 介绍

## ERA-Engine Project

ERA-Engine Project 是一个最初计划用于 [**ERA 游戏**](https://era.miraheze.org/wiki/起始) 的现代游戏引擎的开源项目。我们希望能够创造使开发者、口上和文本作者、非专业玩家都能够轻松开发 ERA 游戏的引擎，并且支持现代程序语言和功能。

## ERA-Engine (Godot)

ERA-Engine (Godot) 是以 [**Godot 游戏引擎**](https://godotengine.org/) 为基础的版本，也是我们当前正在主要开发的版本。

### 支持的语言

- [C#](https://dotnet.microsoft.com/en-us/languages/csharp)
- [GDScript](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/index.html)
