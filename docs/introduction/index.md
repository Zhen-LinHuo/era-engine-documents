# 简介

欢迎阅读 ERA-Engine 文档！

ERA-Engine 是一个基于 Godot 构建的现代化游戏引擎，专为开发 ERA 类游戏而设计。它结合了 C# 核心引擎的高性能与 GDScript 游戏脚本的灵活性，让开发者能够轻松创建拥有丰富文本交互、数值管理和分支叙事的视觉小说风格游戏。
