# ERA-Engine (Godot) 文档

欢迎来到 **ERA-Engine (Godot)** 的文档站！

ERA-Engine 是一个基于 [Godot 游戏引擎](https://godotengine.org/) 的类 ERA 游戏引擎，使用 C# 和 GDScript 编写。本项目旨在为开发者、口上和文本作者、以及非专业玩家提供一个能够轻松开发 ERA 游戏的现代化引擎。
