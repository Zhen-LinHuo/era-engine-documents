# 入门教程

欢迎来到 ERA-Engine 入门教程！本章节将从零开始，手把手带你搭建环境、编写代码，逐步完成一个可运行的小型 ERA 游戏。

无论你是经验丰富的开发者还是刚入门的新手，只要按照教程顺序逐步操作，就能掌握 ERA-Engine 的核心开发流程。

## 教程流程

本教程分为七个步骤，每一步建立在前一步的基础上：

| 步骤 | 教程 | 你将学到 |
|:----:|:-----|:---------|
| 1 | [环境搭建](setup.md) | 安装 Godot (.NET 版本) 与 .NET SDK，克隆并运行 ERA-Engine 项目 |
| 2 | [显示文本](display-text.md) | 使用 `TXT()` 在游戏画面中显示文本内容 |
| 3 | [添加按钮](add-buttons.md) | 使用 `BTN()` 创建交互按钮，实现状态之间的跳转 |
| 4 | [布局容器](layout-boxes.md) | 使用 `BOX()` 组织 UI 元素的排列与布局 |
| 5 | [添加数据](add-data.md) | 使用 CSV 数据文件管理角色、物品等游戏数据 |
| 6 | [完整示例](complete-example.md) | 综合运用所有知识，完成一个包含标题画面、角色选择和主循环的迷你游戏 |

> 📌 **建议**：请严格按照顺序完成每个步骤。每一步的代码都依赖前一步的结果，跳过步骤可能导致代码无法运行。

## 准备工作

在开始之前，请确保你有：

- 一台运行 Windows、macOS 或 Linux 的电脑
- 基本的文本编辑能力（如果你使用过任何编程语言会更容易，但不是必须的）
- 大约 30-60 分钟的时间来完成全部教程

## 完成教程后

完成入门教程后，你将对 ERA-Engine 的核心开发模式有扎实的理解。接下来可以：

- 前往 [开发指南](../guide/index.md) 深入了解引擎的各个子系统
- 阅读 [API 参考](../api/index.md) 查阅完整的 API 列表
- 开始设计你自己的 ERA 游戏！

准备好了吗？让我们从 [环境搭建](setup.md) 开始吧！
