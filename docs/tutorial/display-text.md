# 显示文本

文本是 ERA 游戏的灵魂。在本教程中，你将学习如何使用 `TXT()` 函数在游戏画面中显示文字。

## 打开主状态文件

在 Godot 编辑器的 **FileSystem** 面板中，导航到：

```
Prototype/GdsScript/Flow/main_state.gd
```

双击打开文件。你会看到类似以下的内容：

```gdscript
extends Node

func main_state():
    # This is the entry point of the game.
    # Add your TXT(), BTN() calls here.

    pass
```

`main_state()` 是游戏的入口函数——游戏启动时，ERA-Engine 会自动调用这个函数来渲染第一个画面。

## 你的第一行代码：TXT()

将 `main_state()` 修改为：

```gdscript
func main_state():
    TXT("你好，世界！")
```

修改完成后，保存文件（`Ctrl+S`），然后点击运行按钮（`F5`）。你应该在游戏窗口中看到文字 **"你好，世界！"** 显示在画面上方。

> 🎉 恭喜！你已经成功让 ERA-Engine 显示了一段文本。

## TXT() 函数详解

`TXT()` 是 ERA-Engine 中最基础的函数，用于显示一行文本。

### 基本语法

```gdscript
TXT("要显示的文本")
```

每次调用 `TXT()` 都会在画面上追加一行新的文本。你可以多次调用来显示多行内容：

```gdscript
func main_state():
    TXT("你好，世界！")
    TXT("欢迎来到 ERA-Engine。")
    TXT("这是一段多行文本演示。")
```

运行后，游戏画面中将依次显示三行文本。

### 添加说话者参数

`TXT()` 支持第二个可选参数来指定**说话者的名字**：

```gdscript
TXT("文本内容", "说话者名字")
```

例如，添加一个旁白（narrator）：

```gdscript
func main_state():
    TXT("你好，世界！")
    TXT("欢迎来到这个奇幻的世界。", "旁白")
    TXT("冒险即将开始……", "旁白")
```

运行后，你会看到：

```
你好，世界！
[旁白] 欢迎来到这个奇幻的世界。
[旁白] 冒险即将开始……
```

> 💡 说话者名字的显示格式由引擎的视图系统控制。默认情况下，引擎会在文本前加上 `[说话者]` 前缀。

### 使用换行符

如果你想在单次 `TXT()` 调用中显示多行文本，可以 `\n` 换行符：

```gdscript
TXT("第一行\n第二行\n第三行")
```

## 完整示例

将 `main_state.gd` 的内容替换为以下代码，全面体验 `TXT()` 的功能：

```gdscript
extends Node

func main_state():
    TXT("=== 冒险者公会 ===")
    TXT("你推开厚重的木门，走进了冒险者公会的大厅。", "旁白")
    TXT("壁炉里的火焰噼啪作响，空气中弥漫着麦酒和旧皮革的气味。", "旁白")
    TXT("欢迎光临！有什么可以帮你的吗？", "公会接待员")
```

运行游戏，你将看到一个富有氛围感的场景描述——这正是 ERA 游戏以文字叙事为核心的魅力所在。

## 常见问题

**Q: 文本显示后可以修改吗？**
A: 在一个状态函数内，`TXT()` 调用的文本在状态渲染后无法动态修改。如需根据数据动态生成文本，请参考后续的 [添加数据](add-data.md) 教程。

**Q: TXT() 支持富文本格式吗？**
A: 当前 `TXT()` 支持纯文本。富文本显示可通过自定义 Unit 组件实现。
