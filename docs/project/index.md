## 贡献指南

欢迎为 ERA-Engine 做出贡献！

- **代码仓库**：[ERA-Engine on GitHub](https://github.com/Zhen-LinHuo/era-engine)
- **文档仓库**：[ERA-Engine Documents on GitHub](https://github.com/Zhen-LinHuo/era-engine-documents)
- **问题反馈**：请在 GitHub Issues 中提交 Bug 报告或功能建议
- **文档贡献**：文档使用 MkDocs Material 构建，欢迎提交 Pull Request 完善文档内容

### 本地构建文档

```bash
# 克隆文档仓库
git clone https://github.com/Zhen-LinHuo/era-engine-documents.git
cd era-engine-documents

# 安装依赖并启动开发服务器
pip install mkdocs-material
mkdocs serve
```
